# 🎓 College Event Management System

## 📌 Project Overview

The College Event Management System is a MERN Stack web application developed to manage college events efficiently. It allows users to register, log in, add events, view all events, search events, update event details, and delete events through a simple and user-friendly interface.

## 🚀 Features

* User Registration
* User Login
* Dashboard
* Add New Event
* View All Events
* Search Events
* Edit Event Details
* Delete Events
* Responsive User Interface

## 🛠 Technologies Used

### Frontend

* React.js
* React Router DOM
* Axios
* Bootstrap

### Backend

* Node.js
* Express.js

### Database

* MongoDB
* Mongoose
